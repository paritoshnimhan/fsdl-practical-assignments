# Assignment 05 — Tour Package Booking (Express + MongoDB)

## Aim
To build a server-rendered tour package booking application.

## Technologies Used
- Node.js, Express
- MongoDB (Mongoose)
- EJS templating

## Project Structure
- `app.js` — Express server entry point
- `models/` — Mongoose models (Package, Booking)
- `views/` — EJS templates (index, packages, book, success, partials)
- `public/` — static CSS/images

## Steps to Run
1. `cd source-code`
2. `npm install`
3. Copy `.env.example` to `.env` and set your own `MONGODB_URI` and `PORT`.
4. `node app.js`  (open the printed localhost URL)
