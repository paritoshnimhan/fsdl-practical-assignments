# Assignment 06 — Car Service Booking (Express + MongoDB)

## Aim
To build a car service appointment booking application.

## Technologies Used
- Node.js, Express
- MongoDB (Mongoose)
- HTML/CSS/JS front-end (served from `public/`)

## Project Structure
- `server.js` — Express server entry point
- `models/` — Mongoose model (Appointment)
- `public/` — front-end (index.html, style.css, script.js, logo)

## Steps to Run
1. `cd source-code`
2. `npm install`
3. Copy `.env.example` to `.env` and set your own `MONGODB_URI` and `PORT`.
4. `node server.js`  (open the printed localhost URL)

## Output
See the PDF and screenshots in `output-screenshots/`.
