# Assignment 07 — Student Feedback System (Express + MongoDB)

## Aim
To build a student feedback collection system.

## Technologies Used
- Node.js, Express
- MongoDB (Mongoose)
- HTML/CSS/JS front-end (served from `public/`)

## Project Structure
- `server.js` — Express server entry point
- `models/` — Mongoose model (Feedback)
- `public/` — front-end (index.html)

## Steps to Run
1. `cd source-code`
2. `npm install`
3. Copy `.env.example` to `.env` and set your own `MONGO_URI` and `PORT`.
4. `node server.js`  (open the printed localhost URL)

See the PDF and screenshots in `output-screenshots/`.
